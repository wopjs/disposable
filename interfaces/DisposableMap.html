Interface DisposableMap<TKey>

A Disposable Map is an IDisposable store that manages Disposers and IDisposables with keys.

All Disposers and IDisposables in the Map will be invoked(flush) when the Map is disposed.

A DisposableMap is also a Disposer, which means it can be the dispose method of an IDisposable.

A DisposableMap is also an IDisposable, which means it can be managed by another DisposableMap.

interface DisposableMap<TKey = any> {
    dispose(): void;
    flush(key?: TKey): void;
    has(key: TKey): boolean;
    keys(): IterableIterator<TKey>;
    make<T extends DisposableType>(key: TKey, executor: () => T): T;

        key: TKey,

    remove(key: TKey): undefined | DisposableType;
    set<T extends DisposableType>(key: TKey, disposable: T): T;
    size(): number;
    (): void;
    (): any;
}
